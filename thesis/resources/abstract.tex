\abstractKeywordsEN{Machine learning, Graphs, Neural Networks, Python}

\abstract{
    Praca skupia się na próbie stworzenia modelu sztucznej sieci neuronowej,
    który pozwoliłby z~wysoką pewnością klasyfikować realne rysunki grafów.
    Przeprowadzony został proces projektowania i~budowy odpowiedniego modelu
    z~wykorzystaniem architektury Tensorflow zaimplementowanej jako pakiet
    dla języka Python. Zbiory danych wejściowych zostały przygotowane
    własnoręcznie za pomocą skryptów w języku R za pomocą biblioteki igraph.
    Przeprowadzono szereg testów modeli w celu wyłonienia modelu,
    który osiąga najlepszą dokładność na danych sztucznych oraz rzeczywistych.
}

\abstractEN{
    The thesis focuses on an attempt to create an artificial neural network model,
    which would allow to classify real graph drawings with high confidence.
    The process of designing and building a~suitable model was carried out
    using the Tensorflow architecture implemented as a~package
    for the Python language. The input data sets were prepared
    using scripts in R language, using the igraph library.
    a~number of tests were conducted to select the model
    that achieves the best accuracy on both artificial and real data.
}
